大同清鸟精英教育培训学校

大同市清鸟精英教育培训学校(同教社F91150)于2007年经大同市教育局、民政局批准成立，是从事计算机技能培训、就业培训和国家资格技能鉴定培训，并与多所院校合作开展成人函授教育、网络教育、在职研究生等专业的旗舰教育品牌学校。

学校一直围绕“企业实用性人才紧缺，大中专院校学生就业困难”为着手，成立了自己的就业培训实训基地，以彻底解决用人单位招人难与毕业生找工作难的难题，目前实训基地占地2000平方米，投资100多万元置办实训设备，形成具有高端行业就业技能培训与指导，设备先进，实训内容以企业岗位要求的标准进行专业系统的培训，学校致力于打造一个综合性的短期技能培训+就业输出实训基地。

CSTP人才企业联盟

为更好的服务于企业，并及时将企业用人需求提供给院校，形成信息的双向沟通，由中国软件行业协会发起，中关村软件园、中关村东升科技园、中关村环保科技园、中关村人才市场支持，十余家IT知名企业联合成立“人才企业联盟” 。人才企业联盟由知尚教育负责运营和管理，旨在帮助更多的IT企业找到能支撑企业整体战略和健康发展的合适技术人才，它的运行为中国软件专业人才培养工程教学产品的课程设计提供了真实的企业调研参考依据，并集中了更多、有效的企业用人订单。
